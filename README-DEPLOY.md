# BYR Docs Archive Site

这是从 `byrdocs/byrdocs-archive` 元信息生成的纯静态检索站点。

## GitHub Pages 部署

1. 在 GitHub 新建一个空仓库，例如 `byrdocs-archive-site`。
2. 把本目录里的全部文件上传到仓库根目录，包括 `.nojekyll`。
3. 打开仓库 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main` 和 `/root`，点击 `Save`。
6. 等 GitHub Pages 构建完成后，访问页面给出的 `https://用户名.github.io/仓库名/` 地址。

## 说明

- 本站只索引开源元信息，不重新托管 PDF/zip 文件。
- “打开原始链接”仍指向 `https://byrdocs.org/files/...`。
- 如果原文件服务不可用，站点能搜索，但原始文件链接可能打不开。
